# OpenMemory

| | |
|---|---|
| Repo | [CaviraOSS/OpenMemory](https://github.com/CaviraOSS/OpenMemory) |
| Category | Tier B — MCP, local SQLite |
| First surveyed | 2026-06 |

**What it is.** Local persistent memory for LLM apps (Claude Desktop, Copilot, Codex, etc.): SQLite, hierarchical memory decomposition with a temporal graph on top, zero cloud config.

**Claims under independent test.** Cross-client persistence and the hierarchical-decomposition retrieval quality.

**Quest status.** Surveyed; install & smoke (Phase 1) pending. Benchmark results will appear here as they land.

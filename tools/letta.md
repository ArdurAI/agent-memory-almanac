# Letta (MemGPT)

| | |
|---|---|
| Repo | [letta-ai/letta](https://github.com/letta-ai/letta) |
| Category | Tier A — OS-style memory manager |
| First surveyed | 2026-06 |

**What it is.** Treats the LLM like an operating system managing its own memory: core context (RAM), recall store (recent history), archival store (long-term), with the model paging information in and out.

**Claims under independent test.** Best-for-long-running-agents claim; self-managed paging vs externally-managed retrieval at scale.

**Quest status.** Surveyed; install & smoke (Phase 1) pending. Benchmark results will appear here as they land.
